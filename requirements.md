# Hardware Requirements

Raspberry Pi (used as IoT clients)

One Raspberry Pi acting as Gateway / Federated Learning Server

BeagleBone boards (used as traffic generators for realistic IoT traffic)

Local network connection between devices

# Software Requirements
Operating System

Linux (Raspberry Pi OS / Ubuntu)

Programming Language

C++

Libraries and Tools

* The following packages must be installed before compiling the project.
Install using:

sudo apt update
sudo apt install g++
sudo apt install build-essential
sudo apt install libeigen3-dev
